How It Works

This predictor uses a machine learning model trained on historical Titanic passenger data. The logistic regression algorithm analyzes factors like passenger class, gender, age, and fare to predict survival probability.

Key Survival Factors

  • Gender: Women had higher survival rates
  • Class: First-class passengers survived more
  • Age: Children had better survival chances
  • Cabin: Having a cabin indicated better location
